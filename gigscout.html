Goal

The goal of this project was to develop a fully responsive marketing website for GigScout based on the visual design handed to me in Sketch files. GigScout is an app that allows users to find music shows by location, get tickets for shows, and discover new music.

Process

I was responsible for ensuring that the site adhered to the Desktop and Mobile layouts, but I also ensured that the site would remain visually appealing on a tablet as well.

I used jQuery and CSS3 for hover animations. I also adapted a jQuery plugin (bxslider) for the two sliders with two different types of user control.

    Front End Development Key Deliverables

  • Researching suitable jQuery slider plugins and ulimately choosing and adapting bxSlider jQuery plugin for its diverse options of user control and repsonsivity
  • Making the website fully responsive
  • Understanding the uses of the CSS transition property
  • jQuery and CSS hover animations
