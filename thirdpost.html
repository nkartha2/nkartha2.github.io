Display Modes

Inline
Can't assign width or margins
Element only takes up as much space as the content
Doesn't disrupt the document flow
Naturally inline elements examples: span, b, td, a, img
Block
Element takes up full available width of parent element
Doesn't disrupt the document flow
Naturally block elements examples: h1, p, ul, table
Inline-Block
Has the block mode quality of being able to assign width and margins
Has the inline mode quality of not taking entire width of viewport
Can use vertical align property with this display mode

Font Sizing

Em Units
Em units offer better control. However, em fall into the claws of the compounding effect. The compounding effect happens with nesting elements (elements contained in other elements). Em length unit depends on the font size being used for the element. 1em is equal to the inherent font size. For example, most browsers default the font size to 16px, so 1em is equal to 16px until declared otherwise; therefore, it is not dependent on font-family but on font-size
Rem Units
Rem units are relative to the root element of page so they are not affected by inheritance or the compounding effect. However, they offer less control as ems.

Box Model

Positioning

Relative
Relative positioned elements don't affect the layout of surrounding elements.They are relative to their original position and still within normal document flow.
Fixed
Fixed positioned elements are positioned relative to the browser window. They are out of the normal document flow and don't affect other elements. The github icon is in fixed position.
Absolute
Absolute positioned elements are relative to the first parent that has a relative position. They are out of the normal document flow and don't affect surrounding elements. These elements usually need height defined otherwise the height tends to collapse. This text is in absolute position and the parent (the absolute data term tag) is in relative position.

CSS Layout

Floats
Floats are used to take elements out of the document flow and move them horizontally. Surrounding elements flows or wraps around element. Floats are within the document flow. Therefore, surrounding elements will move up adjacent to the floats (their tendency is to wrap). Another problem that arises when using floats is the collapse of the parent element's height. This column and the right column are positioned using floats and the clearfix property.
Clearfix
The clearfix property forces parent elements to clear children.
