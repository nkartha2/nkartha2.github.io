  • Can you describe the difference between progressive enhancement and graceful degradation?
    • Graceful Degradation: creates an user experience with modern browsers in mind and will create a limited lower experience for older browsers (possibly with less functionality)
    • Progressive Enhancement: creates an user experience with functionality that will work across all browsers in mind and advanced functionality for browsers that can handle it
  • How would you optimize a website's assets/resources?
    • Prioritize semantic HTML5 markup
    • Keep assets light in size
    • I use ImageOptim to optimize images.
    • Load script after page has loaded
    • Use media queries to split stylesheets so that irrelevant stylesheets won't block rendering
    • Async JS so that JS files won't block rendering
    • If CSS minimal, inline styling to optimize website performance
    • Move scripts to the bottom
    • Move CSS to the top
    • Preload images
  • How many resources will a browser download from a given domain at a time?
    • Usually, two requests downloading two files at a time
